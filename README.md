# GreenCommute AI

An AI-powered public transport solution that predicts passenger demand and provides smart recommendations to improve commuting efficiency and encourage public transport usage.

## Features

* Demand Prediction
* Crowd Level Estimation
* Smart Route Recommendations
* Sustainability Insights
* Reward-Based Public Transport Adoption

## Technologies

* HTML
* CSS
* JavaScript
* Agentic AI Workflow (Planned)

## Goal

To make public transportation more efficient, sustainable, and attractive for daily commuters.
